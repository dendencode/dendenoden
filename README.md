#Hello

データ分析と可視化に取り組むエンジニアです。意思決定を支えるダッシュボード構築を目指して、日々データと向き合っています。
不動産から金融まで、様々な分野のデータを扱う中で、正確なデータ処理 と わかりやすい可視化 の両方が、ビジネスの価値につながることを実感してきました。

データアナリストへのキャリアシフトを目指しており、実務を通じたスキル向上に取り組んでいます。

##やっていること

データ抽出 — 公式ソースのAPI活用、効率的なデータ取得

データ品質管理 — クレンジング、検証、改善フロー

可視化・ダッシュボード — 利用者の意思決定を支える設計

分析基盤構築 — 再現性と保守性を意識した開発

#現在の取り組み
実務経験をベースにしたプロジェクトを公開していく予定です。小さなものから、段階的に充実させていきたいと考えています。
データ分析や可視化について、一緒に学べる機会があれば嬉しいです。
